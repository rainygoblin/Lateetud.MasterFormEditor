LEADTOOLS JS Demos

The LEADTOOLS JS and HTML5 Demos demonstrate the use of our SDK with JavaScript and HTML5. Online versions of the demos can be found from the LEADTOOLS HTML5 demos page.

Getting Started

By default, the project should be configured to open this page. This can easily be changed under the project's properties page, navigating to the "Web" tab, and clicking the "Specific Page" radio button so that the project automatically opens at /Apps.

The main sections of this project are as follows:

/Apps

The Apps directory contains multiple client-side applications that use the included API.

/Help

XML Comments, in-code, are automatically compiled and available to view.

/Api

(Links to Ping())

The service routing is all within the "api" namespace - such as api/Test/Ping.
